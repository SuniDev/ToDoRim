  # ToDoRim - 할일 관리, 미리 알림

  ToDoRim은 감성적인 테마를 더한 **할 일 관리 및 시간·위치 알림 설정 앱** 입니다. <br>
  iOS 개발 1년 차에 처음으로 기획부터 개발, 배포까지 혼자 진행한 앱 프로젝트입니다. <br>
  2019년에 MVC 아키텍처로 개발한 이 프로젝트를 2024년에 코드 안정화를 위한 리팩토링 작업을 진행 중입니다. <br><br>
